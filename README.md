## Dependencies

### Gradle (Groovy)
```groovy
implementation 'io.github.tehnewb:Valthorne:1.3.5'
```

### Gradle (Kotlin DSL)
```kotlin
implementation("io.github.tehnewb:Valthorne:1.3.5")
```

### Maven
```xml
<dependency>
    <groupId>io.github.tehnewb</groupId>
    <artifactId>Valthorne</artifactId>
    <version>1.3.5</version>
</dependency>
```
